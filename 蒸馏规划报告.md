## 三、核心产品详细规划

### 3.1 DocuForge - 文档工坊

**合并来源**: docx + pdf + pptx + xlsx + doc-coauthoring

**核心能力提取**:

| 原Skill | 提取能力 | 蒸馏后API |
|---------|---------|----------|
| docx | Word创建/编辑/修订追踪 | `docx.create()`, `docx.edit()`, `docx.trackChanges()` |
| pdf | PDF读取/合并/分割/水印 | `pdf.extract()`, `pdf.merge()`, `pdf.watermark()` |
| pptx | 演示文稿生成/模板 | `pptx.create()`, `pptx.fromTemplate()` |
| xlsx | 电子表格/公式/图表 | `xlsx.analyze()`, `xlsx.chart()` |
| doc-coauthoring | 实时协作 | `collab.start()`, `collab.invite()` |

**统一接口设计**:

```python
class DocuForge:
    """统一文档处理工坊"""
    
    def read(self, path: str) -> Document:
        """智能识别格式，自动解析"""
        
    def create(self, type: str, template: str = None) -> Document:
        """创建文档: 'word' | 'pdf' | 'pptx' | 'xlsx'"""
        
    def convert(self, from_path: str, to_format: str) -> str:
        """格式转换"""
        
    def analyze(self, doc: Document) -> AnalysisReport:
        """文档分析"""
        
    def collaborate(self, mode: str) -> CollabSession:
        """协作模式: 'track' | 'comment' | 'review'"""
```

**定价策略**:
- Free: 基础文档处理 (每月100次)
- Pro ($9.99/mo): 高级格式/批量处理
- Enterprise ($29.99/mo): 协作功能+API

## 七、蒸馏核心方法论

### 7.1 能力提取原则

1. **功能解耦**: 每个Skill拆分为原子能力
2. **接口统一**: 相同类型能力使用统一API
3. **可扩展**: 预留插件机制
4. **原创实现**: 不复制源码，自主实现

### 7.2 蒸馏检查清单

- [ ] 核心能力是否完整保留?
- [ ] 接口是否简洁易用?
- [ ] 是否消除冗余功能?
- [ ] 定价策略是否合理?
- [ ] 是否有差异化竞争力?
